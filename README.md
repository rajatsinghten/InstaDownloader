# 📸 InstaDownloader

A beautiful web tool to download Instagram Reels, Photos, and Videos.

> Built with FastAPI + yt-dlp + vanilla HTML/CSS/JS

## Features

- 🎬 Download Reels, Videos, Photos & Carousels
- 👁️ Preview media before downloading
- 🎨 Stunning dark UI with Instagram-inspired gradients
- ⚡ Fast downloads powered by yt-dlp
- 📱 Fully responsive design
